# Ponto Cego - Landing Page

Landing page estática do evento Ponto Cego.

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Suba este projeto para um repositório no GitHub.
2. Importe o repositório na Vercel.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.

## Usar com v0

Depois que o repositório estiver no GitHub:

1. Abra o v0.
2. Importe o repositório pelo GitHub.
3. Peça para o v0 organizar a landing em componentes, sem alterar copy, layout e identidade visual.
4. Revise o preview.
5. Faça commit/PR pelo próprio v0.
6. A Vercel cria preview deployment automaticamente a cada push.

Prompt sugerido para o v0:

```text
Este é o projeto da landing page Ponto Cego.
Organize o HTML/CSS/JS em uma estrutura limpa e componentizada, mantendo exatamente a copy, identidade visual, responsividade, imagens, contagem regressiva, carrossel e formulário visual.
Não altere textos e não recrie a página do zero.
Apenas melhore organização técnica, performance e manutenção do código.
```
